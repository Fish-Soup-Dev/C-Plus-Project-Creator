# C-Plus-Project-Creator
Simple command-line tool for creating c++ projects with full folder structure and makefile.

## Build Status
| Platform           | Supported |
| :----------------- | :-------- |
| `Linux (x86_64)`   | No        |
| `macOS (arm64)`    | No        |
| `Windows (x86_64)` | Yes       |
